# Mini Market

## Tools:

-   create-react-app
-   react-router
-   redux, react-redux
-   styled-components
-   react-hooks

## Start Project

-   download/clone the project
-   \$ cd picnic-assignment
-   \$ yarn && yarn start
-   open url: localhost:3000

## Available routes

-   `/` & `/list` to display products
-   `/list/product_id` to view a certain product
